# 志愿填报推荐台

一个可直接打开的静态志愿推荐网站，默认内置内蒙古数据。

## 打开

直接打开 `index.html` 即可使用。

## 数据

本地数据文件为 `data/admissions.js`，由 `tools/build-data.js` 生成。

当前构建包含：

- 2025 内蒙古本科批第一次投档最高分/最低分
- 2024 内蒙古本科一批、二批第一次填报最终统计
- 2023 内蒙古本科一批、二批第一次填报最终统计
- 2025/2024/2023 内蒙古一分一段表

重新生成数据：

```powershell
node .\tools\build-data.js
```

或：

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\build-data.ps1
```

## 说明

推荐按 2025、2024、2023 的分数线与一分一段位次做加权计算，固定输出 10 个冲、20 个稳、10 个保。住宿条件和学校环境设施不在官方投档统计中披露，页面保留章程/官网核验字段，不填虚构描述。

页面支持目标地区分类：全部地区、经济发达地区，以及北京、天津、浙江、上海、西安、广州、福建、江苏、其他地区。选择具体地区后会优先推荐该地区院校；若不足 40 个，自动用其它地区补齐，保持冲/稳/保数量不变。
